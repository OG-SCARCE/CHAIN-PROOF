## 📖 Introduction

**ChainProof** is a premium‑grade, browser‑native evidence‑management platform. It lets users capture, hash, transfer, and verify digital artifacts entirely in the client, ensuring zero‑cloud dependency and tamper‑proof audit trails. The UI is built with **React**, **Vite**, **TailwindCSS**, and **Framer Motion**, delivering a sleek, glass‑morphism aesthetic with dynamic micro‑animations.

---

## ✨ Core Features

- **Instant Hashing** – SHA‑256/384/512 hashing performed locally in the browser.
- **Custody Transfer** – Full audit log of every hand‑off with actor, location, and notes.
- **Verification & Alerts** – Re‑run digests on demand; mismatches trigger real‑time alerts.
- **Rich UI** – Glass‑morphism cards, animated terminals, particle‑based backgrounds, and a **gradient‑card hover animation** for the “Steps” section.
- **Responsive Design** – Works on desktop, tablet, and mobile with dark‑mode support.

## ⚙️ Installation & Development

```bash
# Clone the repository
git clone https://github.com/your-username/chainproof.git
cd chainproof

# Install dependencies (Node 20+, npm)
npm install

# Run the development server
npm run dev
```

The app uses **Vite** for fast hot‑module reloading. All styles are pure Tailwind CSS; no external CSS frameworks are required.

---

## 🚀 Usage

1. Open `http://localhost:5173` in your browser.
2. Explore the hero section – notice the animated terminal and the **gradient‑card steps**.
3. Click **Start Evidence Intake** to upload a file and see its hash.
4. Use the navigation header to move between pages.
5. Verify evidence on the **Verify** page; any mismatch will flash a red alert.
